# 🎬 Telegram Video Download Bot

TikTok, Instagram ve X (Twitter) videolarını indirmenize olanak tanıyan Telegram botu.

## Ozellikler

✅ **TikTok** - Videoları yuksek kalitede indirin
✅ **Instagram** - Reels ve videoları indirin
✅ **X (Twitter)** - Videoları indirin
✅ **Kalite Secimi** - 1080p, 720p, 480p secenekleri
✅ **Sadece Ses** - MP3 formatında ses indirme

## Kurulum

```bash
# Gerekli paketleri yukleyin
pip install python-telegram-bot yt-dlp

# Botu baslatin
chmod +x start_bot.sh
./start_bot.sh
```

## Kullanim

1. Bot'a `/start` yazin
2. Video linkini yapistirin
3. Kalite secin veya sadece ses (MP3) secin
4. Video'yu indirin!

## Komutlar

- `/start` - Botu baslat
- `/help` - Yardim menusu

## Gereksinimler

- Python 3.8+
- python-telegram-bot
- yt-dlp

## Not

Ses (MP3) indirmek icin FFmpeg gereklidir.
